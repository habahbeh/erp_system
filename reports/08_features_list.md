# قائمة ميزات نظام المشتريات

## نظرة عامة

نظام المشتريات هو وحدة متكاملة ضمن نظام ERP شامل، يدعم دورة الشراء الكاملة من طلب الشراء حتى الدفع.

---

## الميزات الرئيسية

### 1. إدارة طلبات الشراء (Purchase Requests)

| الميزة | الوصف | الحالة |
|--------|-------|--------|
| إنشاء طلب شراء | إنشاء طلبات شراء داخلية | ✅ |
| الموافقة على الطلبات | سير عمل الموافقة متعدد المستويات | ✅ |
| تحويل إلى أمر شراء | تحويل الطلب المعتمد لأمر شراء | ✅ |
| تتبع حالة الطلب | متابعة مراحل الطلب | ✅ |
| إلغاء الطلبات | إلغاء الطلبات قبل الموافقة | ✅ |

### 2. طلبات عروض الأسعار (RFQ)

| الميزة | الوصف | الحالة |
|--------|-------|--------|
| إنشاء RFQ | إرسال طلبات أسعار للموردين | ✅ |
| إدارة الموردين المدعوين | تحديد الموردين للمشاركة | ✅ |
| استلام عروض الأسعار | تسجيل عروض الأسعار الواردة | ✅ |
| مقارنة العروض | مقارنة الأسعار والشروط | ✅ |
| اختيار أفضل عرض | اعتماد العرض الفائز | ✅ |

### 3. أوامر الشراء (Purchase Orders)

| الميزة | الوصف | الحالة |
|--------|-------|--------|
| إنشاء أمر شراء | إنشاء أوامر شراء للموردين | ✅ |
| الترقيم التلقائي | ترقيم متسلسل (PO/YYYY/XXXXX) | ✅ |
| بنود متعددة | إضافة عدة مواد لنفس الأمر | ✅ |
| الخصومات | خصم نسبة مئوية أو مبلغ ثابت | ✅ |
| الضرائب | حساب الضريبة (شاملة/غير شاملة) | ✅ |
| سير عمل الموافقة | تقديم → مراجعة → موافقة → إرسال | ✅ |
| طباعة أمر الشراء | طباعة PDF | ✅ |

### 4. استلام البضاعة (Goods Receipt)

| الميزة | الوصف | الحالة |
|--------|-------|--------|
| إنشاء محضر استلام | تسجيل البضاعة الواردة | ✅ |
| الاستلام الجزئي | استلام كمية أقل من المطلوب | ✅ |
| الربط بأمر الشراء | ربط الاستلام بأمر الشراء | ✅ |
| فحص الجودة | تسجيل نتائج الفحص | ✅ |
| تحديث المخزون | تحديث تلقائي للكميات | ✅ |

### 5. فواتير المشتريات (Purchase Invoices)

| الميزة | الوصف | الحالة |
|--------|-------|--------|
| إنشاء فاتورة | تسجيل فواتير الموردين | ✅ |
| الترقيم التلقائي | ترقيم متسلسل (PI/YYYY/XXXXX) | ✅ |
| أنواع الفواتير | شراء / مرتجع | ✅ |
| الخصومات | خصم على مستوى الفاتورة | ✅ |
| الضرائب | حساب ضريبة القيمة المضافة | ✅ |
| المطابقة الثلاثية | مطابقة PO ↔ GR ↔ Invoice | ✅ |
| الترحيل المحاسبي | إنشاء قيد محاسبي تلقائي | ✅ |
| إلغاء الترحيل | عكس القيد المحاسبي | ✅ |

### 6. مرتجعات المشتريات (Purchase Returns)

| الميزة | الوصف | الحالة |
|--------|-------|--------|
| إنشاء مرتجع | تسجيل إرجاع البضاعة | ✅ |
| الربط بالفاتورة الأصلية | ربط المرتجع بفاتورة الشراء | ✅ |
| الترقيم المنفصل | ترقيم خاص (PR/YYYY/XXXXX) | ✅ |
| تحديث المخزون | تخفيض الكمية تلقائياً | ✅ |
| القيد المحاسبي | عكس القيد المحاسبي | ✅ |

### 7. عقود الشراء (Purchase Contracts)

| الميزة | الوصف | الحالة |
|--------|-------|--------|
| إنشاء عقد | اتفاقيات شراء طويلة المدى | ✅ |
| تحديد الأسعار | أسعار ثابتة للفترة | ✅ |
| تواريخ الصلاحية | تاريخ بداية ونهاية | ✅ |
| متابعة الاستهلاك | كميات مستهلكة vs متبقية | ✅ |

## الميزات المتقدمة

### 11. دعم متعدد الشركات

- كل شركة لها بياناتها المنفصلة
- ترقيم مستقل لكل شركة
- إعدادات مخصصة لكل شركة

### 12. دعم متعدد الفروع

- كل فاتورة مرتبطة بفرع
- تقارير على مستوى الفرع
- صلاحيات على مستوى الفرع

### 13. دعم متعدد المستودعات

- تحديد مستودع الاستلام
- تتبع المخزون لكل مستودع

### 14. دعم متعدد العملات

- تسجيل الفواتير بأي عملة
- أسعار صرف محدثة
- تحويل للعملة الأساسية

### 15. دعم وحدات القياس

- تحويل بين الوحدات
- وحدة أساسية لكل مادة
- أسعار مختلفة للوحدات

## الخلاصة

نظام المشتريات يوفر حلاً متكاملاً لإدارة دورة الشراء الكاملة مع:
- **42+ ميزة** فعالة
- **12 نموذج** بيانات
- **50+ عرض** (View)
- **90+ رابط** URL
- **تكامل محاسبي** كامل
- **دعم RTL** للعربية
